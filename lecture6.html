How to Deploy Our HTML Code

Lecture 1 Lecture 2 Lecture 3 Lecture 4 Lecture 5 Lecture 6

Day 06

1. Run Locally (Test on Your Computer)

  1. Save your code with a .html extension (example: index.html).
  2. Double-click the file → It will open in your default browser.
  3. This method works only on your computer, not online.

2. Free Hosting Options (Recommended)

(A) GitHub Pages

  1. Create a free account on GitHub.
  2. Create a new repository (example: my-website).
  3. Upload your HTML, CSS, JS files.
  4. Go to Settings → Pages → Select branch → Save.
  5. You will get a link like:
    https://yourusername.github.io/my-website/

(B) Netlify (Drag & Drop)

  1. Sign up at Netlify.
  2. Click Add new site → Deploy manually.
  3. Drag and drop your HTML folder.
  4. You will get a live link like:
    https://your-site.netlify.app

(C) Vercel

  1. Sign up at Vercel.
  2. Connect your GitHub repository.
  3. Click Deploy.
  4. Your website will be live at:
    https://your-project.vercel.app

3. Paid Hosting (Advanced)

  • Buy a domain (e.g., from GoDaddy, Namecheap, Google Domains).
  • Buy hosting (e.g., Hostinger, Bluehost, SiteGround).
  • Upload your HTML files using FTP (FileZilla) or cPanel.
  • Your site will be live at www.yoursite.com.

4. Best Method for Students

If you want to practice and share projects, use GitHub Pages or Netlify because they are free and professional.

Conclusion

✔ For practice: Run locally on your computer.
✔ For portfolio and sharing: Use GitHub Pages or Netlify.
✔ For professional websites: Buy a domain and hosting.
