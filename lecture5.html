First Principle: The Web is More Than Text and Images

A webpage should deliver any kind of content, not just static text and images. Earlier, browsers needed plugins like Flash, QuickTime, or Silverlight for video/audio, which was inefficient and insecure.

The Core Problem

How to embed video/audio natively, in a standardized way, without plugins? We need a universal system across browsers and devices.

The Logical Solution: The HTML5 <video> and <audio> Tags

Dedicated tags to embed and control media.

Part 1: Basics - Getting a Video on the Page

<video src="my-awesome-video.mp4"></video>

Problem: Without controls, it only shows a frame, not a player. Solution: Add controls attribute. 

<video src="my-awesome-video.mp4" controls></video>

Part 2: Intermediate - Control with Attributes

  • width & height: Set player size.
<video src="video.mp4" width="640" height="360" controls></video>
  • autoplay: Starts automatically (often needs muted).
<video src="video.mp4" autoplay muted controls></video>
  • loop: Repeats automatically.
  • poster: Thumbnail image.
<video src="video.mp4" poster="images/video-thumbnail.jpg" controls></video>

Part 3: The <audio> Tag

Works like video but only for sound.

<p>Listen:</p>
<audio src="audio/theme-song.mp3" controls></audio>

Part 4: Advanced & Professional

Problem #1: Format Support

Different browsers support different formats. Use multiple <source>.

<video controls poster="images/thumbnail.jpg">
  <source src="videos/my-video.webm" type="video/webm">
  <source src="videos/my-video.mp4" type="video/mp4">
  Sorry, your browser doesn't support embedded videos.
</video>

Problem #2: Accessibility

Use <track> for subtitles/captions.

<video controls>
  <source src="video.webm" type="video/webm">
  <source src="video.mp4" type="video/mp4">

  <track src="captions_en.vtt" kind="captions" srclang="en" label="English">
  <track src="subtitles_es.vtt" kind="subtitles" srclang="es" label="Español">
</video>

Two Embedding Methods

Example 1: Robust (using <source>)

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

Example 2: Simple (using src)

<video src="a.mp4" controls></video>
