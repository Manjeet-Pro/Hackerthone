HTML: Hyper Text Markup Language

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

Key Concepts:

  • HyperText: The ability to create links that connect web pages to one another, forming a connected "web" of information.
  • Markup Language: You use "tags" to surround your content, giving meaning and structure. This process is called "marking up" the plain text.
  • First Website: https://info.cern.ch/hypertext/WWW/TheProject.html

Basic HTML Elements

1. Headings (<h1> to <h6>)

Headings are tags used to define titles and subtitles on your page, creating a logical hierarchy.

<h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3>

<h1> is the most important heading, typically used once per page.

2. Paragraph (<p>)

Used to group sentences and blocks of text.

<p>This is the first paragraph.</p> <p>This is a second paragraph.</p>

3. Horizontal Rule (<hr>)

Defines a thematic break in content, often displayed as a horizontal line.

<p>This is the first topic.</p>
<hr />
<p>This is a completely different topic.</p>

4. Line Break (<br>)

Inserts a single line break without starting a new paragraph.

<p>221B Baker Street<br>London, England</p>

5. Lists (<ul>, <ol>, <li>)

Used for ordered and unordered lists.

Example unordered list:

  • Tea Bag
  • Water
  • Sugar
  • Milk

Example ordered list:

  1. Boil water in a kettle.
  2. Add tea bag, sugar and milk.
  3. Keep boiling it for 5 minutes.
  4. Serve the tea by pouring it into a cup.

6. Anchor Tag (<a>)

Used to create clickable links.

<a href="https://www.coderarmy.in/">Visit Coder Army</a>

Use the target attribute to open links in a new tab:

<a href="https://www.coderarmy.in/" target="_blank">Visit Coder Army</a>

7. Image Tag (<img>)

Used to embed images with required src and alt attributes.

<img src="https://cdn.britannica.com/16/234216-050-C66F8665/beagle-hound-dog.jpg" height="300px" alt="Beagle Hound Dog" />
